### Notes

- This is an Android app using Jetpack Compose, LiteRT, and Gemma 4 E2B.
- Unit tests should be placed alongside the code files they test (e.g., `MyClass.kt` and `MyClassTest.kt`).
- Use `./gradlew test` for unit tests and `./gradlew connectedAndroidTest` for instrumented tests.
- RAG knowledge base is pre-built offline and bundled as SQLite + JSON assets in the APK.
